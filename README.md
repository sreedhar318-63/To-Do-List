Vanilla JavaScript To-Do List App
A simple, front-end only To-Do list application built using Vanilla JavaScript, HTML, and CSS. This project allows users to add, manage, and remove tasks dynamically without any external libraries or frameworks.
Features
Add Tasks: Easily add new tasks using an input field and "Add" button (or by pressing Enter).
Mark as Complete: Click on a task's text to toggle its completion status (toggles a strikethrough style).
Delete Tasks: Remove individual tasks using the associated "Delete" button.
Responsive Design: Basic styling provides a clean, user-friendly interface that works well on different screen sizes.
Technologies Used
HTML5: Provides the structure of the application.
CSS3: Handles the styling and visual presentation.
Vanilla JavaScript: Implements all the interactive and dynamic functionality.
Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.
Prerequisites
You need a code editor (like Visual Studio Code) and a modern web browser (like Google Chrome).
Installation and Usage
Clone or Download: Download the project files (index.html, style.css, script.js) to a folder on your computer.
Open with Live Server (Recommended):
If you are using VS Code, install the Live Server extension.
Right-click on the index.html file in your editor and select "Open with Live Server". This will automatically launch the app in your browser and refresh the page as you make changes.
Manual Opening:
Alternatively, simply double-click the index.html file in your file explorer. It will open in your default web browser. Note that this method requires you to manually refresh the browser page after making any code changes.
